Esse site tem objetivo de expor meu trabalho, tanto como cientista de dados, quanto como analista de GIS e pesquisador científico. Sinta-se livre para acessar os links e entrar em contato comigo pelas redes sociais ou WhatsApp.

Ciência de Dados

Utilizando ferramentas de análise de dados, bancos de dados relacionais e não-relacionais, Machine Learning e Cloud, é possível a identificação de padrões e predição de valores futuros em diversos tipos de conjuntos de dados.

Geoprocessamento

Dados espaciais são peça-chave em grande parte das atividades econômicas do mundo globalizado. O geoprocessamento é a ferramenta aplicada para utilizar esses dados em benefício das pessoas e das organizações.

Pesquisa científica

O método científico é o caminho pelo qual são realizadas novas descobertas, gerando inovação e desenvolvendo o processo tecnológico. Para colocar a ciência em prática, o pensamento analítico é a base de tudo.
