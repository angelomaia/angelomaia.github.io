Projects

CadêBuffet?

Developed during the individual phase of TreinaDev by Campus Code, CadêBuffet is a web application designed to connect potential clients with companies that offer buffet services. The system includes several features, such as company and user registration, listing and creation of buffets and event types, as well as a scheduling system with pricing, chat functionality, and customer reviews.

It was built using Ruby on Rails, Bootstrap, and JavaScript, following Test-Driven Development practices and automated testing with RSpec and Capybara.

PagueAluguel

Developed during the Team Experience phase of TreinaDev by Campus Code, PagueAluguel is one of two applications that together form a connected system via API integration. It consumes data from the CondoMinions application and functions as a payment management system for condominiums. Features include the registration of shared utility bills, condominium fees, one-time charges, and rental agreements—ultimately generating invoices for residents of the managed buildings.

Built with Ruby on Rails, Bootstrap, Stimulus.js, JavaScript, RuboCop, SimpleCov, and asynchronous background jobs. The project followed Test-Driven Development practices and includes automated tests using RSpec and Capybara.

Rebase Labs

Developed during the job interview process for Rebase, this project consisted in a lightweight Ruby application using Sinatra. It is a dockerized system with back-end, front-end, and database services. The app is built to handle data related to patients, doctors, and exam results. It has an endpoint to receive the upload of csv files, and a structured interface to show the exam results with pagination and search results by exam token. The upload system has validation of file format and data structure.

The overall aim of this project was to test whether the applicants had the ability to adapt to an unknown framework and do the job within a limited timeframe.
